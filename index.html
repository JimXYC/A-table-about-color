Ways to set up colors!

Type Example Description
Name color=blue Directly providing the name of the color.
Hexadecimal Value color=#FFFFFF Providing the two-digit hexadecimal value of red, green and blue
RGB value color=rgb(39, 65 10) Providing the decimal value for red, green and blue
Hue, Saturation and Lightness color=hsl(240, 100%, 50%) Providing the hue value, the saturation of the color and the lightness of the color
